# 📘 JavaScript & DOM 기초 문제 정리

## ✅ 1. JavaScript의 표준 명세 이름은 무엇인가요?

**정답: c) ECMAScript**

**해설:**
ECMA 재단에서 정의한 표준 명세의 이름이 ECMAScript이며, JavaScript는 이 표준을 따른 구현입니다.

---

## ✅ 2. 다음 중 변수명으로 사용할 수 없는 것은?

**정답: c) let**

**해설:**
`let`은 변수 선언 키워드이며 **예약어**라서 변수명으로 사용할 수 없습니다.

---

## ✅ 3. 변수 선언 시 기본적으로 사용을 권장하는 키워드는?

**정답: c) const**

**해설:**
`const`는 재할당을 막아 안전한 코드를 작성할 수 있게 해줍니다. 필요한 경우에만 `let`을 사용합니다.

---

## ✅ 4. 웹 페이지(Document)를 객체로 표현한 모델은?

**정답: d) DOM**

**해설:**
DOM(Document Object Model)은 웹 문서를 객체 트리 구조로 표현한 API입니다.

---

## ✅ 5. CSS 선택자와 일치하는 첫 번째 요소를 반환하는 메서드는?

**정답: d) querySelector()**

**해설:**
`querySelector()`는 제공된 CSS 선택자와 일치하는 **첫 번째 요소**를 반환합니다.

---

## ✅ 6. CSS 선택자와 일치하는 모든 요소를 NodeList로 반환하는 메서드는?

**정답: c) querySelectorAll()**

**해설:**
`querySelectorAll()`은 선택자와 일치하는 모든 요소를 **NodeList 형태로 반환**합니다.

---

## ✅ 7. 요소에 클래스를 추가하는 classList의 메서드는?

**정답: c) add()**

**해설:**
`element.classList.add()`는 해당 요소에 새로운 클래스를 추가합니다.

---

## ✅ 8. 요소의 속성 값을 설정하는 메서드는?

**정답: d) setAttribute()**

**해설:**
`setAttribute(name, value)`는 지정한 속성 값을 추가하거나 변경합니다.

---

## ✅ 9. 요소의 순수한 텍스트 콘텐츠를 조작하는 속성은?

**정답: b) textContent**

**해설:**
`textContent`는 HTML 태그를 무시하고 순수한 텍스트만 가져오거나 설정합니다.

---

## ✅ 10. 새로운 HTML 요소를 메모리에 생성하는 메서드는?

**정답: b) createElement()**

**해설:**
`createElement('tag')`는 새로운 요소를 생성하며, DOM에 추가하기 전까지 메모리에 존재합니다.

---

## ✅ 11. 생성한 요소를 특정 부모 요소의 마지막 자식으로 추가하는 메서드는?

**정답: c) appendChild()**

**해설:**
`appendChild()`는 부모 요소의 **마지막 자식으로 요소를 삽입**합니다.

---

## ✅ 12. var 키워드의 특징이 아닌 것은?

**정답: c) 블록 스코프**

**해설:**
`var`는 **함수 스코프**를 가지며, let/const가 블록 스코프를 가집니다.
