
# 📘 핵심 키워드 정리

| 개념               | 설명                 | 예시                          |
| ---------------- | ------------------ | --------------------------- |
| **원시 자료형**       | 값이 직접 저장되는 불변 데이터  | Number, String, Boolean 등   |
| **일치 연산자(===)**  | 값과 타입이 모두 같은지 비교   | `'1' === 1 // false`        |
| **for...of 반복문** | 반복 가능한 객체의 값을 반복   | `for (const i of arr) {}`   |
| **함수 표현식**       | 변수를 통해 함수를 정의      | `const sub = function() {}` |
| **화살표 함수**       | 함수 표현식을 간결하게 작성    | `const arrow = () => {}`    |
| **나머지 매개변수**     | 여러 인자를 배열로 받는 매개변수 | `function (...args) {}`     |
| **전개 구문(...)**   | 반복 가능한 항목을 펼치는 문법  | `myFunc(...numbers)`        |

---

# 📘 JavaScript의 데이터 타입

JavaScript의 데이터 타입은 크게 **두 가지**이다.

---

## 🔹 1. 원시 자료형 (Primitive type)

* 값이 변수에 **직접 저장됨**
* 종류:

  * Number
  * String
  * Boolean
  * null
  * undefined
* **불변(immutable)**
* 다른 변수로 전달할 때 **값 복사**

---

## 🔹 2. 참조 자료형 (Reference type)

* 변수에는 **값이 아닌 객체의 메모리 주소**가 저장됨
* 종류:

  * Object
  * Array
  * Function
* **가변(mutable)**
* 다른 변수에 할당하면 **주소가 복사**되어 원본과 사본이 서로 영향을 줌

---

# 📘 원시 자료형의 종류

* **Number**

  * 정수 + 실수 모두 표현하는 숫자 타입

* **String**

  * 텍스트 데이터
  * `+`로 문자열 결합 가능

* **템플릿 리터럴**

  * 백틱(`) 문법
  * `${변수}` 형태로 문자열 내부에 변수 삽입 가능

* **null & undefined**

  * null: 개발자가 의도적으로 "값 없음"을 나타낼 때 사용
  * undefined: 값이 할당되지 않았을 때 시스템이 자동으로 부여

* **Boolean**

  * true / false 논리값

---

# 📘 비교 연산자

## 🔸 동등 연산자 (==)

* 비교 시 **타입을 변환**하여 비교
* 예측 불가한 결과가 나올 수 있어 주의

## 🔸 일치 연산자 (===)

* **값 + 타입 모두 비교**
* 타입 변환 없음
* 예측 가능성이 높아 사용 권장

---

# 📘 논리 연산자

### ✔ AND: `&&`

### ✔ OR: `||`

### ✔ NOT: `!`

**단축 평가(short-circuit evaluation) 지원**

---

# 📘 반복문

## 🔹 for...in

* 객체의 **key(속성 이름)** 순회
* 배열에는 적합하지 않음
  (인덱스를 문자열로 변환하며 순서를 보장하지 않음)

---

## 🔹 for...of

* 반복 가능한 객체의 **값(value)** 순회
* 배열, Map, Set 등에 사용
* 객체(Object)에는 직접 사용할 수 없음

---

# 📘 함수

## 🔸 함수 정의 방법

### ✔ 함수 선언식

```js
function funcName() {}
```

* 호이스팅 영향을 받음
  → 선언 이전에도 호출 가능

---

### ✔ 함수 표현식

```js
const funcName = function() {}
```

* 호이스팅되지 않음
* 코드 예측 가능성이 높아 권장

---

### ✔ 화살표 함수

```js
const arrow = () => {}
```

* 함수 표현식을 더 간결하게 작성
* 본문이 한 줄이면 `{}`와 `return` 생략 가능

---

# 📘 함수의 매개변수

## 🔹 1. 기본 매개변수

```js
function hello(name = 'Anonymous') {}
```

전달되지 않은 인자를 대비한 기본값 설정

---

## 🔹 2. 나머지 매개변수 (...args)

```js
function func(...args) {}
```

정해지지 않은 수의 인자들을 배열 형태로 받음

---

## 🔹 3. 전개 구문 (...)

```js
myFunc(...numbers)
```

배열이나 반복 가능한 객체를 **펼쳐서 전달**할 때 사용
