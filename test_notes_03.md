# 📘 핵심 키워드 정리

| 개념              | 설명                 | 예시                            |
| --------------- | ------------------ | ----------------------------- |
| **객체(Object)**  | 키-값 쌍의 데이터 집합      | `{ key: 'value' }`            |
| **메서드(Method)** | 객체 속성에 정의된 함수      | `user.greeting()`             |
| **this 키워드**    | 함수를 호출한 객체를 가리킴    | `console.log(this.name)`      |
| **구조 분해 할당**    | 객체 속성을 변수에 할당하는 문법 | `const { name } = user`       |
| **JSON**        | 데이터를 표현하는 문자열 형식   | `JSON.stringify(obj)`         |
| **콜백 함수**       | 다른 함수에 인자로 전달되는 함수 | `numbers.forEach(myFunc)`     |
| **forEach 메서드** | 배열의 모든 요소를 순회      | `names.forEach((name) => {})` |

# 📘 객체(Object)

* 객체는 **키(key)**와 **값(value)**으로 구성된 속성들의 집합을 저장하는 자료형
* 점(.) 또는 대괄호([]) 표기법으로 속성 접근/추가/수정/삭제 가능

## ✔ 메서드와 this

* **메서드**: 객체 속성에 할당된 함수
* **this 키워드**: 메서드를 호출한 객체를 가리킴
* 일반 함수의 this → 전역 객체(window)를 참조
* 화살표 함수는 자신만의 this를 가지지 않음

  * 상위 스코프의 this를 그대로 사용

👉 그래서 **객체 메서드 내부의 중첩된 함수에서 this가 달라질 때 화살표 함수가 유용함**

---

# 📘 추가 객체 문법

## ✔ 구조 분해 할당

```js
const { a, b } = obj
```

* 객체 속성을 변수로 쉽게 꺼내 쓰는 문법

## ✔ 전개 구문(...)

```js
const newObj = { ...oldObj }
```

* 객체의 속성을 펼쳐서 새로운 객체 생성(얕은 복사)

## ✔ Optional Chaining (?.)

```js
user.address?.street
```

* 중첩된 객체를 안전하게 접근
* 중간 값이 null/undefined이면 에러 대신 undefined 반환

# 📘 JSON (JavaScript Object Notation)

* 데이터를 표현하기 위한 **문자열 형식**
* 자바스크립트 객체와 구조는 비슷하지만 **문자열 기반**이라는 점이 다름

### ✔ 주요 메서드

* `JSON.stringify()` : 객체 → JSON 문자열 변환
* `JSON.parse()` : JSON 문자열 → 객체 변환

---

# 📘 배열(Array)

* 순서가 있는 데이터의 집합 저장 자료구조

## ✔ 배열 메서드 (원본 배열 직접 수정)

* `push()` : 배열 끝에 요소 추가
* `pop()` : 배열 끝 요소 제거
* `unshift()` : 배열 앞에 요소 추가
* `shift()` : 배열 앞 요소 제거

---

# 📘 배열(Array) - Array Helper Methods

배열을 순회하며 특정 작업을 수행하는 고차 함수들로, **콜백 함수**를 인자로 받는다.

## ✔ 콜백 함수

* 다른 함수의 인자로 전달되어
  특정 작업이 끝난 후 외부 함수 내부에서 실행되는 함수

---

## ✔ forEach()

* 배열의 각 요소에 대해 콜백 실행
* **반환값 없음(undefined)**
* 원본 배열을 변경하지 않음

---

## ✔ map()

* 배열의 각 요소에 콜백 함수를 적용한 결과를 모아
  **새로운 배열을 반환**
* 원본 배열은 변경되지 않음
