## 📘 코드 실행 결과 작성

9. **다음 코드의 실행 결과를 작성하시오.**

   ```python
   class Test:
       count = 0

       def __init__(self):
           Test.count += 1

       @classmethod
       def get_count(cls):
           return cls.count

   a = Test()
   b = Test()
   print(Test.get_count())
   ```
