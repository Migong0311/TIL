### 9. 다음 코드 실행 결과로 옳은 것을 고르시오.

```python
for i in range(5):
    if i == 2:
        continue
    print(i, end=" ")
```

a) 0 1 2 3 4
b) 0 1 3 4
c) 1 2 3 4
d) 0 1 2

---

### 10. 다음 중 break 문이 실행되는 조건은?

```python
nums = [1, 3, 5, 7, 10, 11]
for n in nums:
    if n % 2 == 0:
        break
    print(n)
```

a) 1, 3, 5, 7
b) 1, 3, 5, 7, 10, 11
c) 10, 11
d) 아무것도 출력되지 않는다.
