
# 📘 JavaScript 심화 개념 문제 정리 (정답 + 해설 포함)

## 1. this 키워드에 대한 설명으로 올바른 것은?

### 🔹 선지

a) 항상 전역 객체(window)를 가리킨다.
b) 함수가 선언될 때 가리키는 대상이 정해진다.
c) 메서드를 호출한 객체를 가리킨다.
d) 화살표 함수는 자신만의 this를 가진다.

### ✅ 정답: **c) 메서드를 호출한 객체를 가리킨다.**

### 📘 해설

* `this`는 **함수가 호출되는 방식**에 따라 동적으로 결정된다.
* 메서드를 호출하면 `this`는 **그 메서드를 호출한 객체**를 가리킨다.
* 화살표 함수는 자신만의 this를 가지지 않고 **외부 렉시컬 환경의 this를 사용**한다.

## 2. 키와 값의 변수명이 같을 때 사용하는 문법은?

### 🔹 선지

a) 단축 속성
b) 계산된 속성
c) 구조 분해 할당
d) 전개 구문

### ✅ 정답: **a) 단축 속성**

### 📘 해설

* 객체 정의 시 `name: name`처럼 키와 변수명이 같다면 → `name`으로 축약 가능

  ```js
  const name = '홍길동';
  const obj = { name }; // 단축 속성
  ```

## 3. 객체의 속성을 분해하여 변수에 할당하는 문법은?

### 🔹 선지

a) 단축 속성
b) 계산된 속성
c) 구조 분해 할당
d) Optional Chaining

### ✅ 정답: **c) 구조 분해 할당**

### 📘 해설

* 구조 분해 할당은 객체 속성을 변수로 쉽게 추출하는 문법

  ```js
  const { age, name } = user;
  ```

## 4. 객체를 JSON 형식의 문자열로 변환하는 메서드는?

### 🔹 선지

a) JSON.parse()
b) JSON.stringify()
c) Object.toJSON()
d) Object.toString()

### ✅ 정답: **b) JSON.stringify()**

### 📘 해설

* `JSON.stringify()`는 JS 객체 → JSON 문자열로 변환
* 반대로 JSON 문자열 → JS 객체는 `JSON.parse()`.

## 5. 배열의 **가장 마지막에 요소를 추가**하는 메서드는?

### 🔹 선지

a) pop()
b) shift()
c) unshift()
d) push()

### ✅ 정답: **d) push()**

### 📘 해설

`push()`는 배열 끝에 새 요소 추가 + 새로운 길이 반환

## 6. 배열의 **가장 앞에서 요소를 제거**하는 메서드는?

### 🔹 선지

a) pop()
b) shift()
c) unshift()
d) push()

### ✅ 정답: **b) shift()**

### 📘 해설

`shift()`는 배열의 첫 요소 제거 후 제거된 값을 반환

## 7. 콜백 함수에 대한 설명으로 가장 적절한 것은?

### 🔹 선지

a) 함수가 자기 자신을 다시 호출하는 함수
b) 다른 함수의 인자로 전달되는 함수
c) 여러 개의 함수를 하나로 묶는 함수
d) 반드시 비동기적으로 작동하는 함수

### ✅ 정답: **b) 다른 함수의 인자로 전달되는 함수**

### 📘 해설

콜백 함수는 특정 작업이 끝난 후 실행시키기 위해 **다른 함수에 인자로 전달하는 함수**.

## 8. 배열의 각 요소에 대해 함수를 실행하지만, 아무것도 반환하지 않는 메서드는?

### 🔹 선지

a) map()
b) filter()
c) forEach()
d) reduce()

### ✅ 정답: **c) forEach()**

### 📘 해설

`forEach()`는 배열 요소마다 함수를 실행하지만 **undefined를 반환**한다.

## 9. 배열의 모든 요소에 함수를 적용한 결과로 **새로운 배열**을 만드는 메서드는?

### 🔹 선지

a) map()
b) forEach()
c) for…of
d) Object.keys()

### ✅ 정답: **a) map()**

### 📘 해설

`map()`은 각 요소에 함수를 적용하여 **새 배열 생성**.

## 10. 객체의 모든 키(key)를 배열로 반환하는 메서드는?

### 🔹 선지

a) Object.keys()
b) Object.values()
c) Object.entries()

### 📘 해설

* d) 옵션은 존재하지 않는 잘못된 문법

## 11. Optional Chaining(?.)을 사용하는 주된 목적은?

### 🔹 선지

a) 속성이 없으면 에러를 발생시키기 위해
b) 중첩 객체의 속성을 에러 없이 안전하게 접근하기 위해
c) 여러 함수를 연달아 호출하기 위해
d) 중첩된 객체의 속성을 삭제하기 위해

### ✅ 정답: **b) 중첩 객체의 속성을 에러 없이 안전하게 접근하기 위해**

### 📘 해설

Optional chaining은 다음과 같은 오류를 방지한다.

```js
user?.address?.city
```

→ 속성이 없어도 undefined 반환으로 에러 방지

## 12. 클래스로부터 새로운 객체 인스턴스를 생성할 때 사용하는 키워드는?

### 🔹 선지

a) create
b) instance
c) new
d) this

### ✅ 정답: **c) new**

### 📘 해설

```js
const user = new User();
```
